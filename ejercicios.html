Ejercicios de Variables

Ejercicio 1: Declará una variable nombre con tu nombre y mostralo usando print().

Ejercicio 2: Creá dos variables, edad y anio_actual, y calculá en qué año naciste.

Ejercicio 3: Definí una variable producto, otra precio y otra cantidad. Calculá el total a pagar.

producto = "Cuaderno"
precio = 120.50
cantidad = 3
total = precio * cantidad
print("Producto:", producto)
print("Total a pagar:", total)

Ejercicios de Condicionales

Ejercicio 6: Escribí un programa que diga si una persona es mayor de edad (mayor o igual a 18 años).

edad = 20
if edad >= 18:
    print("Sos mayor de edad")
else:
    print("Sos menor de edad")

Ejercicio 7: Verificá si un número es par o impar usando el operador módulo %.

Ejercicio 8: Escribí un programa que diga si una persona puede acceder a un beneficio: es_mayor_edad y tiene_permiso deben ser True para que acceda.

es_mayor_edad = True
tiene_permiso = False

if es_mayor_edad and tiene_permiso:
    print("Puede acceder al beneficio")
else:
    print("No puede acceder")

Ejercicio 9: Escribí un programa que clasifique una nota:

  • 10: Excelente
  • 7-9: Muy bien
  • 4-6: Aprobado
  • 0-3: Reprobado

Ejercicio 10: Completá el siguiente código para que verifique si un usuario puede entrar a un sitio solo si tiene más de 13 años y acepta los términos:

edad = 14
acepta_terminos = True

if edad > 13 and acepta_terminos:
    print("Acceso permitido")
else:
    print("Acceso denegado")
