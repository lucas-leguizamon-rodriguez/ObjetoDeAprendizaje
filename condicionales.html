Condicionales en Python

Las condicionales permiten que un programa tome decisiones evaluando si una condición es verdadera o falsa. En Python, usamos if, elif y else para esto.

Sintaxis básica

if condicion:
    # bloque de código si la condición es verdadera
else:
    # bloque de código si la condición es falsa

Condición con elif

nota = 7

if nota >= 9:
    print("Excelente")
elif nota >= 6:
    print("Aprobado")
else:
    print("Reprobado")

¿Qué es print()?

La función print() se utiliza para mostrar información en pantalla. En el caso de las condicionales, permite visualizar qué camino tomó el programa según la condición.

Comparadores más usados

  • == igual a
  • != distinto de
  • > mayor que
  • < menor que
  • >= mayor o igual que
  • <= menor o igual que

Buenas prácticas

  • Usá identación consistente (4 espacios) después de if, elif o else
  • Escribí condiciones claras y fáciles de leer
  • No olvides los dos puntos : al final de la línea condicional

Ejemplo completo con entrada de datos

edad = 17

if edad >= 18:
    print("Sos mayor de edad")
else:
    print("Sos menor de edad")

Ejercicio práctico

Completá el siguiente código que analiza la edad y condición de socio para ver si la persona accede a un beneficio:

edad = 20
es_socio = True

if edad >= 18 and es_socio:
    print("Puede acceder al beneficio total")
elif edad >= 18 and not es_socio:
    print("Accede con inscripción previa")
else:
    print("No puede acceder")

¿Qué hace este código? Evalúa dos condiciones: si la persona es mayor de edad y si es socio. Dependiendo del resultado, imprime un mensaje distinto.
