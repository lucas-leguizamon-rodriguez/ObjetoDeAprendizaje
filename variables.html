Variables en Python

Una variable es un espacio de memoria que se utiliza para almacenar un dato o valor que puede cambiar durante la ejecución del programa. En Python, las variables se crean automáticamente al asignarles un valor usando el signo igual (=).

¿Cómo se asigna una variable?

La sintaxis básica para crear una variable es:

nombre_variable = valor

Ejemplos:

edad = 25

nombre = "Lucía"

peso = 55.8

activo = True

Tipos de datos comunes

Tipo Descripción Ejemplo
int Números enteros 10, -3
float Números decimales 3.14, 0.5
str Cadenas de texto "Hola", 'Python'
bool Valores booleanos True, False

Reglas para nombrar variables

  • Deben comenzar con una letra o guion bajo (_)
  • No pueden comenzar con un número
  • No pueden contener espacios ni caracteres especiales
  • Se recomienda usar snake_case: palabras separadas por guion bajo (edad_usuario)

Ejemplos con explicación:

# Guardamos un número entero
años = 20

# Guardamos una cadena de texto
nombre = "Carlos"

# Guardamos un número decimal
altura = 1.75

# Guardamos un valor booleano
estudiante = True

Más ejemplos útiles

# Calcular el área de un rectángulo

base = 5

altura = 10

area = base * altura

print("Área del rectángulo:", area)
# Concatenar cadenas de texto

nombre = "Sofía"

apellido = "García"

nombre_completo = nombre + " " + apellido

print("Nombre completo:", nombre_completo)
# Operaciones con booleanos

mayor_edad = True

tiene_permiso = False

puede_entrar = mayor_edad and tiene_permiso

print("¿Puede entrar?:", puede_entrar)

¿Qué es print()?

La función print() se utiliza para mostrar información por pantalla. Es útil para mostrar mensajes, resultados de operaciones, o verificar valores de variables mientras programás.

print("Hola mundo")

Buenas prácticas

  • Usar nombres descriptivos: nombre_completo, precio_total
  • Evitar nombres como x o dato (salvo en ejemplos simples)
  • Usar siempre snake_case para mejorar la legibilidad

Ejercicio práctico

Completá el siguiente código para que imprima un resumen de una persona con su nombre completo, edad, saldo en su cuenta y si puede acceder a un beneficio especial:

nombre = "María"
apellido = "Pérez"
edad = 30
saldo = 1520.75
es_cliente_vip = True

nombre_completo = nombre + " " + apellido

print("Nombre completo:", nombre_completo)
print("Edad:", edad)
print("Saldo disponible:", saldo)

¿Qué hace este código? Muestra el nombre completo, edad y saldo de una persona. Además, verifica si tiene acceso a beneficios especiales en base a un valor booleano usando una estructura condicional if.
